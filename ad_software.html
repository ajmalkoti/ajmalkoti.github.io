Software


  • FDWave package. This MATLAB package is developed for seismic wave simulation in 2D-Acoustic/Elastic/Viscoleastic, as well as for 3D-HTI/VTI/fully-anisotropic media. Documentation for FDwave2.0


  • ANCC HVR package. This Python package is developed for numerical estimation of H/V ratio of the cross-correlation obtained from ambient noise.


  • URG package. This MATLAB package is developed for upscaling Acoustic/Elastic wave equation using Renormalization group theory.


  • FARMCA package. A MATLAB package for estimating the permeability of a rock using monte carlo method.


  • GALog package. Inverting the well log data using GA algorithm for petrophysical parameters.


  • GASeis package. FORTRAN 95 package for inverting the 1D seismic data uisng GA algorithm.
